# Online Device Repair Tracking & Management System

🚀 A sponsored mini project developed for **Snehpreet Enterprises**.

This web-based platform simplifies device repair management by enabling customers to track repair status online and helping service providers manage repair requests efficiently.

🔗 **Live Demo:** [https://online-device.vercel.app/](https://online-device.vercel.app/)
